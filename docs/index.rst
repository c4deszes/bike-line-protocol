LINE is a bus type protocol that uses a single controller, it's based on the LIN protocol used in
automotive products and extends the frame length and possible

The project goes beyond just specification, an implementation for C projects is available under the
``src`` folder, complete with a Python library that allows communicating with devices and there's
also a code generator that can be integrated into CMake projects.
